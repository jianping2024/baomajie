## 当前状态

- 当前阶段：第一版基础工程已开始搭建
- 首发市场：葡萄牙
- 首发语言：简体中文，预留葡萄牙语和其他欧洲语言
- 后端已确定：NestJS、Fastify Adapter、Prisma、PostgreSQL
- 管理后台已确定：独立Next.js应用
- 当前工作区未包含旧网站代码、数据库备份或数据字典
- 第一版不依赖旧数据库，使用现有网站公开页面作为数据来源
- 当前仓库已加入 `apps/web` 和 `apps/api` 的工程骨架

## 已确认的第一版决策

1. 在线支付：不包含，保留后续扩展能力。
2. 站内消息：包含基础版本。
3. 服务商入驻：允许个人和企业。
4. 用户联系方式：公开，但提供用户控制、反爬和风险提示。
5. 旧用户及密码：暂不迁移。
6. 旧数据库或导出API：暂不提供。
7. 第一版重点：抓取现有网站公开数据，完成问题入口型首页和公开详情页。
